# Code of Conduct

This is a personal portfolio project, but issues, discussions, and pull
requests are open to anyone. A short standard applies:

- Be respectful and constructive in issues, PRs, and discussions.
- No harassment, personal attacks, or discriminatory language.
- Stay on topic — this repo covers the site's code and content, not general
  grievances.

Reports of unacceptable behavior can be sent to the maintainer at
[Mkhonzasenzo525@gmail.com](mailto:Mkhonzasenzo525@gmail.com). Reports will be
reviewed and handled at the maintainer's discretion, which may include
warning, blocking, or removing content from a contributor.

This is a lightweight adaptation of the
[Contributor Covenant](https://www.contributor-covenant.org/), scaled to the
size of this project.
